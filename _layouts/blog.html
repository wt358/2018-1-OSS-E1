How to install

이 페이지에서는 phantombot을 설치하는 방법에 대해서 설명할것 입니다.

version java 8.0 을 설치하기

Version8.0 이상의 상위 버전이라 할지라도 다운그레이드 시켜야 한다.

https://www.java.com/en/download

위의 URL을 이용해서 java 8.0을 다운로드 받아오길 바란다.Java 8.0을 다운로드 시킨 다음 꼭 version 확인을 하기 바란다.(밑의 URL을 사용해도 된다.)

https://www.java.com/en/download/installed.jsp

13 September 2016 this should show Version 8 Update 101 의 java 8.0을 사용하는 것이 가장 이상적이다

PhantomBot 설치하기

https://phantombot.tv/#download

위의 링크에서 먼저 PhantomBot을 다운로드 시켜야한다.Stable version은 일반적인 사용을 목적으로 한 버전이다. The nightly는 실험적인 버전이며, 소스은 개발자들을 위한 버전이다. 안정 버전을 다운로드하면 PhantomBot-x.x.zip 파일 하나 (x.x는 버전 번호가 표시됨)를 얻을 수 있으며 원하는 위치로 압축을 풀어야 합니다. 압축을 풀면 PhantomBot-x.x 폴더에 PhantomBot을 실행하는 데 필요한 파일이 포함됩니다.

각자의 챗봇 계정 설정

Bot account가 있을 경우, 이 과정은 생략하고 넘어가라. Bot account을 만들려면 Twitch에서 새 계정을 만들어야 한다. 다른 로그인을 하고 있는 경우, 로그 아웃하면 이 과정을 실행할 수 있습니다. 이후에 다음 링크를 사용하여 새계정을 만드십시오.

https://www.twitch.tv/signup

처음 실행 시, 요구 사항

처음 실행하는 경우, 이 과정에서는 몇 가지 세부 사항을 설정해주어야 한다. • Bot Account Name • Bot OAuth Token • Caster OAuth Token • Which channel to join [가입할 채널] - Bot Account Name는 미리 알고 있어야만 한다. 이후의 작업은 Bot Account Name을 알고 있다는 전제 아례에서 진행된다.

Bot OAuth Token

Bot OAuth Token을 얻으려면 bot에 대해 생성한 계정으로 twitch에 로그인해라. 이 token은 챗봇을 Twitch와 연결시켜준다. 여기까지 진행되었다면, 다음 url로 이동하라 https://twitchapps.com/tmi . 연결한 이후엔 권한 부여를 클릭하고 코드를 안전한 위치로 복사하십시오. 코드는 다음과 같이 보일 것입니다.

oauth:123456789
